
# Blog Preview Card

A simple and responsive Blog Preview Card built using only HTML and CSS.

## Preview

This project is a clean card UI design featuring:
- Blog image
- Category badge
- Publish date
- Title and description
- Author section
- Soft shadow and rounded corners

## Built With

- HTML5
- CSS3
- Flexbox

## Features

- Responsive layout
- Modern card design
- Clean typography
- Custom shadow and spacing

## What I Practiced

- Flexbox alignment
- Card layout design
- CSS shadow
- Border radius
- Spacing and typography
- Responsive centering
